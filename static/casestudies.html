Four case studies

This section presents how four case studies has been modelled in WRITE Knowledge Base. In particular, each case study represents an example artwork belonging to one of the WRITE collections (respectively Contemporary Visual Art, Decorative and Applied Art, Graffiti Art and Performance Art).

This case study illustrates "Ars Divina" Grattifo by ERICTIN from Kwayin Clan. The artwork presents some descriptive metadata (e.g. title, description, subjects, graffiti style, location, date of creation). Additionally, the artwork creator (EricTin) is instance of the class wd:1215627 (Person). Even if WRITE KB stores metadata about artists and involved agents, they are not described in this case study for the sake of simplicity. For this reason, the case study presents only the connection between EricTin and the crew to which he belongs to - the Kwayin Clan marked as an Organisation (wd:Q43229). Moreover, two Calli-writing units belongs to the artwork ("CWU 1 Shengong Yijiang (神工意匠)" and CWU 2 Shengong Yijiang (天际征鸿)") and both of them are stored as inviduals of the class write:CalliWritingUnit along with their respective metadata (e.g. metadata for artistic analysis - colors, tools, materials; metadata for linguistic analysis - character form, script style, calligraphic line, transcription...). Both unit contents has been based over two already existing literary works (individuals of the class wd:7725634). Literary works are stored in WRITE KB along with some basic descriptive metadata, but again for the sake of simplicity they are not recorded in this case study.

This case study illustrate the Musicalligraphy performance by Luo Qi and Collegium Musicum Almae Matris. The performance has been modelled as an event with the involved actors, date, location, implied musical instruments etc. Additionally, during the performance event, the calligrapher and performer Lou Qui paints some calligraphic contents with ink and brush in a panel made of paper. The panel, considered in this project a pieces of art itself, is represented as a Contemporary Visual Artworks (instance of the class wd:108923606) along with its respective descriptive metadata (e.g. title, dimensions, creator etc). The performance, and therefore the produced panel, has in common the same Calli-writing unit (instance of the class write:CalliWritingUnit). As in the previous case study, the unit is recorded with its respective metadata (e.g. metadata for artistic analysis - colors, tools, materials; metadata for linguistic analysis - character form, script style, calligraphic line, transcription...))

The case study shows how the produced Contemporary Visual Artwork is strictly connected with the performance in which it has been created by the object property write:creates-artwork.

In WRITE KB this musicalligraphy performance is connected with two panels (and therefore two corresponding units). For the sake of simplicity, this case study omits the second panel, but it can still be found in the Knowledge Base.
