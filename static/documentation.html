WRITE data model


Chinese tradition, culture and art represent a certain challenge when it comes to the application of Digital Humanities' practices. Even though DH literature on Linked Open Data in the cultural heritage is extremly rich, we can still observe the lack of studies and publications that tackle data models used for representation of Chinese calligraphy, both artisticly and linguistically.

Thus, WRITE data model contains both reused and made ad hoc classes and properties, in order to fully represent the complex phenomenon of Chinese contemporary calligraphy.

WRITE Knowledge Base is populated with elements from 14 classes (owl:Class), 10 of which have been reused from Wikidata, and other 4 were created on purpose. This documentation illustrates these 14 classes along with their respective properties (owl:ObjectProperty and owl:DataTypeProperty).

Each of the four WRITE Collections, namely Visual Arts, Graffiti Art, Decorative & Applied Arts and Performing Arts, are represented with a dedicated class. It was important to preserve the diversity between them, in order to emphasise their individual peculiar elements.

For example, when describing artworks belonging to the class Graffiti, it is important to record particularities related specifically to this type of artwork, such as the graffiti style or a chinese visual element. On the other hand, WRITE data model seeks to express the complexity of Performing arts, given that the realisation of a performance can employ numerous external artworks, as well as different actors in several roles. Furthermore, Decorative & Applied arts can offer insight into the commercial and economical aspect of these artworks, while Visual artworks in this data set demonstrate both interconnection with other artworks and modern tendencies related to digital arts.

Data entry process is being done using Omeka S platform. Here, the term Resource Template is always referred to as a synonym for owl:Class. These resource templates as fully customisable and can be enriched with various metadata. Once filled and saved, a resource template becomes an item of the dataset, as an individual belonging to a class. Visit OmekaS website for more information.

In this documentation, each Class is presented with a brief introduction and a corresponding image graphically summarising all the possible connections with the other Classes/Resource Templates (owl:Class, via owl:ObjectProperty), as well as the metadata fields associated with the class in question.

Performing Arts resource template (owl:Class) represents Performing arts items. The individuals of this class constitute one of the four collections of the WRITE KB.

Series of Creative Works resource template (owl:Class) has been added in order to establish an explicit connection between items of the four collections with the work serie they may belong to and to record some descriptive metadata about the group of artworks

Person resource template (owl:Class) represents the complex network of agents intended as singles that operates in the context of this project: authors, artists, performers, calligraphers, dancers, musicians, calligraphers, curators, organisers and so on. Each template represents:

  • Basic metadata to describe the persion biographical information (e.g. name, date of birth, occupation..)

Categorising Contemporary Calligraphy seemed a hard task. This means that no explicit definition of Calligraphic Unit has been performed. Instead we defined general Calli-writing units (owl:Class, which corresponds with some artwork particular content unit) with few metadata fields which define traditional calligraphy characteristics aiming to dispose the Knowledge Base for an analysis aiming to lead to the definition of how much Traditional Calligraphy can be seen in Contemporary Calligraphy.

Literary Work resource template (owl:Class) has been added to store information between Calli-writing units and literary production they may refer to. This part does need some additional work for example by adding metadata about period of creation, type of production etc.

Traditional Artwork is a resource template (owl:Class) that represents basic metadata about the artworks made before 1980 that are somehow related to the contemporary artworks of the 4 WRITE collections.

Life Event resource template (owl:Class) has been added to store information about artist and their academic, personal, working life (it belongs mainly to the Socio-Political Aspect). NB. An interesting aspect for the future can be to look at European Influences over artists (where did they were born, did they work, studied, can they talk in English or other languages etc).

Document is a resource template (owl:Class) reprenting basic metadata about publications and archival docuements which are somehow related to the artist and their production
