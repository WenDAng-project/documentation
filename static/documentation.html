The model documentation

WRITE Knowledge Base stores WRITE data is a triple-fashion. The KB is populated with elements from 11 reused or created on purpose Classes (owl:Class). The documentation below illustrates the 11 classes along with their respective properties (owl:ObjectProperty and owl:DataTypeProperty).

The four Collections (Contemporary Visual Arts, Graffiti, Decorative & Applied Arts, Performing Arts) has been represented with four different Classes aiming to preserve the diversity between the described elements. For example, when describing Graffiti is important to record the graffiti style of the item; Performances are considered as to be at the same time events and artworks. For this reason, they can imply other artworks in the making of the performance as well as different actors is several roles; when describing Decorative & Applied the commercial aspect of these products needs to be stored etc.

The term "Resource Template" is always referred as a synonym of owl:Class. Such term has been used during the data entry acrivity in OmekaS. OmekaS is constituted of customizable forms to fill with metadata. Each resource template can be associated with an owl:Class. This means that each filled resource template becomes an item of the dataset, as an individual belonging to a class. Visit OmekaS website for more information.

Each Class is presented with a brief introduction and a corresponding image graphically summarising all the properties associated with the class (on the right) and the possible connections (via owl:ObjectProperty) with the other Resource Templates (owl:Class).

Contemporary Visual Art resource template (owl:Class) represents contemporary fine arts items. The individuals of this class constitute one of the four collections of WRITE KB.

Decorative and Applied Artwork resource template (owl:Class) represents contemporary fine arts items. The individuals of this class constitute one of the four collections of WRITE KB.

Graffiti resource template (owl:Class) represents contemporary graffiti art items. The individuals of this class constitute one of the four collections of WRITE KB.

Performance resource template (owl:Class) represents contemporary performance art items. The individuals of this class constitute one of the four collections of WRITE KB.

Series of Creative Works resource template (owl:Class) has been added in order to establish an explicit connection between items of the four collections with the work serie they may belong to and to record some descriptive metadata about the group of artworks

Person resource template (owl:Class) represents the complex network of agents intended as singles that operates in the context of this project: authors, artists, performers, calligraphers, dancers, musicians, calligraphers, curators, organisers and so on. Each template represents:
  • The relations between a person and the resources of the KB
  • Basic metadata to describe the persion biographical information (e.g. name, date of birth, occupation..)

Organisation resource template (owl:Class) represents the network of agents intented as groups that operates in the context of this project: collectives, crews, dance companies, event organisers, insitutions and so on. Each template represents:
  • The relations between an organisation and the resources of the KB
  • Basic metadata to describe the organisation descriptive metadata (e.g. name, period of acitivity, members etc)

Exhibition resource template (owl:Class) represents the exhibitions to which at least one of the described artworks took part of

Categorising Contemporary Calligraphy seemed a hard task. This means that no explicit definition of Calligraphic Unit has been performed. Instead we defined general Calli-writing units (owl:Class, which corresponds with some artwork particular content unit) with few metadata fields which define traditional calligraphy characteristics aiming to dispose the Knowledge Base for an analysis aiming to lead to the definition of how much Traditional Calligraphy can be seen in Contemporary Calligraphy.

Literary Work resource template (owl:Class) has been added to store information between Calli-writing units and literary production they may refer to. This part does need some additional work for example by adding metadata about period of creation, type of production etc.

Life Event resource template (owl:Class) has been added to store information about artist and their academic, personal, working life (it belongs mainly to the Socio-Political Aspect). NB. An interesting aspect for the future can be to look at European Influences over artists (where did they were born, did they work, studied, can they talk in English or other languages etc).

Document is a resource template (owl:Class) reprenting basic metadata about publications and archival docuements which are somehow related to the artist and their production
